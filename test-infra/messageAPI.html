Message API

属性：

title '设置message 标题内容'
body '设置message 正文内容'
type: '可选样式' 'info' 默认 样式蓝色 ，例如：

...

$('#myMessage).message()

...
'warning' 样式黄色 ，例如：

...

$('#myMessage).message({

type:''warning''

})

...
'error' 样式红色 ，例如：

...

$('#myMessage).message({

type:'error'

})

...
'complete' 样式绿色 ，例如：

...

$('#myMessage).message({

type:'complete'

})

...
cls: '显示按钮' 1 默认 ，1代表“确定”例如：

...

$('#myMessage).message({

cls: 1

})

...
2 2代表“是” ，例如：

...

$('#myMessage).message({

cls: 2

})

...
4 4代表“取消” ，例如：

...

$('#myMessage).message({

cls: 4

})

...
8 8代表“否” ，例如：

...

$('#myMessage).message({

cls: 8

})

...
1|2|4 选择三个按钮 ，例如：

...

$('#myMessage).message({

cls: 1|2|4

})

...
time: '显示倒计时' 0 默认 0 ，不设置倒计时，如果设置变量大于0，则该变量为倒计时的时间。

方法：

'show' 调用 show 方法，显示 ， 参数e, fn,添加回调函数，
例如：当e=1,该回调函数为“是”按钮的回调函数
'hide' 调用 hide 方法，隐藏
'title' 调用 setTitle 方法，参数（messageTitle）例如：

...

$('#myMessage).message('title');

...
'body' 调用 setBody 方法，参数（body）例如：

...

$('#myMessage).message('body');

...
‘touch’ 调用 touch 方法，参数（type, fn）例如：

...

$('#myMessage).message('touch', 'sure', function (e, o) {

$('#myMessage).message('hide');

})

...
