Counter API

属性：

type 设置盒子排列方式， 0为水平 num-horizontal 1为垂直 num-vertical，默认为0。
size 设置盒子大小尺寸，sm为小尺寸，不设置为大尺寸。默认不设置。
max 设置数量盒子可改变最大值。
min 设置数量盒子可改变最小值，默认为1。
data-state='disabled' 调用counter 方法的默认元素如果定义data-state='disabled'属性，则数量盒子不可操作。

事件：

getValue 触发getValue 事件，可得到当前数量盒子的值。

方法：

setVal 调用 setVal 方法，可设置数量盒子的值， 设置参数为指定的值，如果不设置则默认为0。
operation 调用 show 方法，取消数量盒子data-state='disabled' 属性，将该盒子变为可操作。
