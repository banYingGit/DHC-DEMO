Process API

属性：

element '设置创建process的目标容器'
max '设置process 的最大值,默认为100'
active '设置值为1时，process 显示为100%'
type '设置值为1时，process 显示为弹出层'

事件：

'start' 进度条加载前触发start事件
'process' 进度条加载中触发process事件,可得到‘当前值’，‘最大值’，‘百分比值’
'end' 进度条加载后触发end事件,默认end内部私有事件为隐藏弹出进度条。
'open' 弹出进度条加载后自动关闭，当再次触发open事件时,可打开进度条

方法：

'value' 调用 value 方法，添加参数值，显示当前进度
