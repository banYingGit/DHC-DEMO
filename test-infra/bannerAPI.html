banner API

属性：

focusPosition 【焦点对齐】'参数left、 right、 center，分别为焦点左对齐、右对齐、居中对齐，默认为不显示'
focusStyle 【焦点样式】'参数img、 text、 point，分别为显示缩略图、标题、点点，默认为点点'
animation: 【动画效果】'参数fade、slide，分别为淡入淡出、滑动，默认为淡入淡出'
interval： 【动画间隔时间】'参数以ms 为单位计算，默认为5000，即5s'

事件：

loopEnd 触发 loopEnd 事件可在轮播图播放一圈后添加回调函数
end 触发 end 事件 可在轮播图播放结束后添加回调函数

方法：

next 调用next 方法 可跳到下一张
prev 调用prev 方法 可跳到上一张
num 调用num 方法，参数‘数字’， 可跳到指定一屏
autoPlay 调用autoPlay 方法 可执行自动播放
stopPlay 调用stopPlay 方法 可执行停止播放
