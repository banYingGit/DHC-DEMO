数量盒子 NumberBox

1.默认样式

数量盒子组件样式在.num 基础上追加样式.num-horizontal 为水平排列，上追加样式.num-vertical为垂直排列。

  • Demo
  • Html

  • <div class="num num-horizontal">

    <input name="" type="text" class="num-input">

    <input type="button" class="num-add" value="+">

    <input type="button" class="num-reduce" value="-">

    </div>

  • Demo
  • Html

  • <div class="num num-vertical">

    <input name="" type="text" class="num-input">

    <input type="button" class="num-add" value="+">

    <input type="button" class="num-reduce" value="-">

    </div>

2.盒子尺寸

数量盒子在基础尺寸样式上追加样式 .num-sm 为小尺寸盒子。

  • Demo
  • Html

  • <div class="num num-horizontal num-sm">

    <input name="" type="text" class="num-input">

    <input type="button" class="num-add" value="+">

    <input type="button" class="num-reduce" value="-">

    </div>

  • Demo
  • Html

  • <div class="num num-vertical num-sm">

    <input name="" type="text" class="num-input">

    <input type="button" class="num-add" value="+">

    <input type="button" class="num-reduce" value="-">

    </div>

3.盒子状态

3.1数量盒子在input上追属性 disabled 为不可点击状态。

  • Demo
  • Html

  • <div class="num num-horizontal">

    <input name="" type="text" class="num-input" disabled>

    <input type="button" class="num-add" disabled value="+">

    <input type="button" class="num-reduce" disabled value="-">

    </div>


    <div class="num num-vertical num-sm">

    <input name="" type="text" class="num-input">

    <input type="button" class="num-add" value="+">

    <input type="button" class="num-reduce" value="-">

    </div>
