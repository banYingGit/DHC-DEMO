按钮 Button

1.基本样式

1.1 默认样式

在要应用按钮样式的元素上添加 .btn，再设置相应的颜色。

  • Css
  • Demo
  • Html
  •   默认颜色 主色按钮 次色按钮 默认边框 主色边框 次色边框 文字链接
    class .btn-default .btn-primary .btn-secondary .btn-default-alt .btn-primary-alt .btn-secondary-alt .btn-link
  • 文字链接

  • <button class="btn btn-default">默认样式</button >

    <button class="btn btn-primary">主色按钮</button >

    <button class="btn btn-secondary">次色按钮 </button>

    <button class="btn btn-default-alt">默认边框 </button>

    <button class="btn btn-primary-alt">主色边框 </button>

    <button class="btn btn-secondary-alt">次色边框</button >

    <a href="#" class="btn-link">文字链接</a>

1.2 圆角按钮

在默认样式的基础上添加 .btn-round

  • Demo
  • Html

  • <button class="btn btn-default btn-round">默认圆角</button >

2、按钮状态

禁用样式 在默认样式的基础上添加 .disabled

  • Demo
  • Html

  • <button class="btn btn-default disabled">按钮禁用</button >

    <button class="btn btn-default-alt btn-round disabled" >按钮禁用</button>

3、按钮尺寸

按钮尺寸为内联样式，每个按钮都有最小值，在其基础上设置大小为padding值。

  • Css
  • Demo
  • Html
  •   超小按钮 小按钮 默认大小 最大按钮
    class .btn-xs .btn-sm .btn-md .btn-lg

  • <button class="btn btn-default btn-xs">超小按钮</button >

    <button class="btn btn-default btn-sm">小按钮</button >

    <button class="btn btn-default">默认大小</button >

    <button class="btn btn-default btn-lg">大按钮</button >

4、块级按钮

添加class .btn-block

  • Demo
  • Html

  • <div style="width:350px; height:35px;">

    <button class="btn btn-default btn-block">块级按钮 </button>

    </div>
