3.复选框

a、input 属性Id 必须与 Lable 属性 For 的值对应一样

b、表单禁用样式 无需加样式 仅加属性 disable

3.1默认样式

调用复选框默认样式 在复选框父级添加样式 check-group

  • Demo
  • Html
  • 复选一 复选二 复选三 复选四

  • <div class="check-group">

    <input type="checkbox" name="group1" id="group1_1" checked="" >

    <label for="group1_1"></label>

    <input type="checkbox" name="group1" id="group1_2" >

    <label for="group1_2"></label>

    <input type="checkbox" name="group1" id="group1_3">

    <label for="group1_3"></label>

    <input type="checkbox" name="group1" id="group1_4" disabled >

    <label for="group1_4" ></label>

    </div>

    <input type="checkbox" name="group2" id="group2_2" >

    <label for="group2_2" data-val="复选2"> </label>

  • Html

  • <input name="testForm" type="text" class="form-box error" placeholder="请输入内容" data-error="top">

    <label class="error"><i class="icon-warning-1"> </i> 校验提示</label>

    <input name="testForm" type="text" class="form-box error" placeholder="请输入内容" data-error="right">
