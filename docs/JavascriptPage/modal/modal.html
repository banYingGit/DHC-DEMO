查看API模态框 Modal

1.基本使用

在使用过程中需按组织好你自己 HTML 代码，当此断代码调用modal时即将该HTML 追加到 modal-body 中，作为模态框内容区域。

  • Demo
  • Html
  • javaScript

  • modalBody 已经存在内容，直接调用。

    modalBody 已经存在内容，直接调用。

  • <div id="modalA"><h3 class="p-10">我是内容区域,不加载load 数据</h3></div >

  • <script>

    $(function () {

    var sureBtn = $('<button class="btn btn-md btn-default" >保 存</button>'),

    cancelBtn = $('<button class="btn btn-md btn-default-alt" data-role="close">取 消</button>');

    $('#modalA').modal({

    title: '我是标题',

    url: 'modalData.html'

    })

    .on('shown',

    function () {

    console.info("打开之后---")

    })

    .on('show',

    function () {

    console.info("打开之前---")

    })

    .on('hidden',

    function () {

    console.info("关闭之后------")

    })

    .on('hide',

    function () {

    console.info("------关闭之前")

    });

    $('#openA').click(function () {

    $('#modalA').modal('button', [sureBtn, cancelBtn]).modal('show')

    });

    })

    </script>

2.动态创建

在使用过程没有任何 HTML 代码，当动态创建一dom 节点元素并且调用modal时，即可创建。模态框内容可以通过设置url 数据连接，此数据为 modal-body 内容。

  • Demo
  • javaScript

  • 动态创建modal

  • <script>

    $(function () {

    var opt = $('<div >').modal(),

    addBtn = $('<button class="btn btn-md btn-default" >增 加</button>'),

    delBtn = $('<button class="btn btn-md btn-default-alt" data-role="close">删 除</button>');

    $('#openB').click(function () {

    opt.modal('show')

    .modal('load', 'modalData.html')

    .modal('title', '我是第二个标题')

    .modal('button', [addBtn, delBtn])

    });

    })

    </script>
